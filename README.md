# Todo List API

A RESTful API built with Flask and SQLite.

## Setup
1. Clone the repo
2. Create a virtual environment: `python -m venv venv`
3. Activate it: `.\venv\Scripts\Activate.ps1`
4. Install dependencies: `pip install -r requirements.txt`
5. Run: `python app.py`

## Endpoints
- POST /register
- POST /login
- GET /todos
- POST /todos
- GET /todos/<id>
- PUT /todos/<id>
- DELETE /todos/<id>
